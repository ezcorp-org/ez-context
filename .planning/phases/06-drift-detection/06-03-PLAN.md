**src/commands/drift.ts:**

Follow the exact pattern of `generateAction` in `src/commands/generate.ts`. Create `driftAction` with:

1. Signature: `async function driftAction(pathArg: string, options: { file?: string; threshold?: string }): Promise<void>`

2. Flow:
   a. Resolve `projectPath` from `pathArg`
   b. Start spinner: `ora("Loading context files...").start()`
   c. Create bridge: `const bridge = await createBridge(projectPath)`
   d. **Index check (CRITICAL)**: `if (!(await bridge.hasIndex(projectPath)))` -> `spinner.fail("No search index found")`, print `chalk.red("Run 'ez-context generate' or 'ez-search index .' first to create an index.")`, `process.exit(1)`
   e. **Resolve files**: If `options.file` given, use that path (resolve relative to cwd). Otherwise auto-detect from `CANDIDATE_FILES = ["CLAUDE.md", "AGENTS.md", ".cursorrules", "CONTEXT.md"]` -- check each in `projectPath`, collect existing ones.
   f. If no files found: `spinner.fail("No context files found")`, print `chalk.red("No CLAUDE.md, AGENTS.md, or .cursorrules found. Use --file to specify one.")`, `process.exit(1)`
   g. **Extract claims**: For each file, `readFile(filePath, "utf-8")` then `extractClaims(content, filePath)`. Collect all claims.
   h. Update spinner: `Analyzing ${claims.length} claims...`
   i. **Score claims**: `scoreClaims(allClaims, bridge, (done, total) => { spinner.text = \`Checking claim ${done}/${total}...\` })`
   j. **Build report**: For each source file, filter its scored claims and `buildDriftReport(sourceFile, fileScoredClaims)`. Also build an aggregate report if multiple files.
   k. **Render and print**: `spinner.succeed(...)`, then for each report print `renderDriftReport(report)` to console. Use chalk for the health score: green if >= 70, yellow if >= 40, red if < 40.
   l. Wrap everything in try/catch: `spinner.fail("Drift analysis failed")`, `console.error(chalk.red(message))`, `process.exit(1)`

3. Import from: `../core/ez-search-bridge.js`, `../core/drift/claim-extractor.js`, `../core/drift/claim-scorer.js`, `../core/drift/report.js`, plus `node:path`, `node:fs/promises` (readFile), `node:fs` (existsSync), `ora`, `chalk`.
