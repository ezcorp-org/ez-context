**generate.ts** -- Update `generateAction` to:

1. Accept `format` in the options object (string, comma-separated).
2. Parse the format string: split by comma, trim whitespace, validate each value against the `OutputFormat` type. If any value is invalid, print an error listing valid formats and exit(1). Use a Set for deduplication.
3. Pass `formats` array to `EmitOptions` when calling `emit()`.
4. Update the dry-run output to show each rendered format (iterate `result.rendered` entries instead of hardcoded claudeMd/agentsMd).
5. Update the success output to list all files written (already works via `result.filesWritten`, just verify the log message makes sense for non-claude formats).
6. Update spinner text to mention format count when non-default: e.g., "Generating 3 context files..." vs "Generating context files...".

Validation function (keep it simple):
```typescript
const VALID_FORMATS: OutputFormat[] = ["claude", "agents", "cursor", "copilot", "skills", "rulesync", "ruler"];

function parseFormats(raw: string): OutputFormat[] {
  const formats = [...new Set(raw.split(",").map(s => s.trim()))];
  const invalid = formats.filter(f => !VALID_FORMATS.includes(f as OutputFormat));
  if (invalid.length > 0) {
    throw new Error(`Invalid format(s): ${invalid.join(", ")}. Valid: ${VALID_FORMATS.join(", ")}`);
  }
  return formats as OutputFormat[];
}
```
